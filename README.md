# RJ.TwoShot | Premium Cinematic Studio Portfolio

A minimalist, high-end creative film studio portfolio for **RJ.TwoShot** (inspired by Apple, Awwwards, and Linear aesthetics).

## Tech Stack
- **Frontend**: React, React Router, TailwindCSS, Framer Motion, GSAP, Lucide Icons, ReactPlayer.
- **Backend**: Node.js, Express, MongoDB (Local/Mock support).
- **Storage**: Cloudflare R2 object storage integration.

## Key Features
- **Portfolio-First Homepage**: Instantly showcases collections under Categories with premium grayscale-to-color transition cards.
- **Cinematic Player**: Immersive full-screen player with ambient blur backdrops, related video carousels, and responsive controls.
- **Admin Dashboard**: Secure management system (`/admin`) for videos and categories with Cloudflare R2 uploads.
